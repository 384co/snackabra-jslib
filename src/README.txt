
In this directory, we have a "tsc" view of the world.

Just do:

$ tsc


================================================================
Utilities

$ searchLibs.sh STRING

this will search all the libraries used by tsc (for the 'es2022'
setting). Currently it looks in the global installation directory
for version 4.8.3:

/opt/homebrew/Cellar/typescript/4.8.3/libexec/lib/node_modules/typescript/lib
